References & Data Availability

United States Sentencing Commission. (2024). Fiscal Year 2024 Individual Offender Datafile. Washington, DC: Office of Research and Data.
Access
Publicly available via USSC.gov. No restricted use agreement required for replication.

Analysis Stack
Python 3.11 Scikit-Learn Statsmodels
Repository
Full source code, including data cleaning, model training, and fairness auditing scripts, is available in the project GitHub repository.
Reproducibility
Methodology
All analyses use fixed random seeds (42) for deterministic results. An automated verification script is included to re-run all statistical tests and validate findings against the raw data.
